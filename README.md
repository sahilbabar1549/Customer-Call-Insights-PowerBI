## 📊 Call Center Performance Dashboard – Power BI Project
## 📌 Project Overview

The Call Center Performance Dashboard is an interactive Power BI project designed to analyze and monitor call center operations.The dashboard provides business insights into call volume, agent efficiency, resolution rates, customer satisfaction, and overall service performance.This project helps management make data-driven decisions to improve customer service quality and operational efficiency.

 ## 🎯 Problem Statement

Call centers generate large amounts of daily data, but without proper visualization, it is difficult to:

Monitor agent performance

Track resolved and unresolved calls

Measure customer satisfaction

Identify performance trends

Improve service efficiency

This dashboard solves these challenges by transforming raw call data into meaningful insights.

## 📊 Dataset Description

The dataset used in this project includes information such as:

Call ID

Call Date

Agent Name

Call Duration

Call Status (Resolved / Unresolved)

Customer Rating

Department / Category

Issue Type

The data was cleaned and transformed using Power Query Editor in Power BI.

## 📈 Dashboard Features & Analysis
1️⃣ Total Calls Overview

Displays total number of calls received.

Helps understand call volume trends.

Shows daily or monthly call distribution.

2️⃣ Call Resolution Analysis

Percentage of resolved vs unresolved calls.

Identifies areas where service needs improvement.

Tracks resolution rate performance.

3️⃣ Agent Performance Analysis

Compares agents based on:

Number of calls handled

Resolution rate

Average handling time

Helps identify top performers.

Highlights agents who may need training.

4️⃣ Customer Satisfaction Analysis

Displays average customer rating.

Shows rating distribution.

Identifies relationship between resolution and satisfaction.

5️⃣ Call Duration Insights

Average handling time.

Identifies long-duration calls.

Helps optimize response efficiency.

## 🛠 Tools & Technologies Used

Data Visualization Techniques
